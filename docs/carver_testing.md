Carver Testing Commands
=======================

Mega Ravines
------------
- Spectator mode underground Y=-128..64.
- /locatebiome minecraft:plains
- /tp @s <x> 100 <z>
- Look for wide ravines.

Sinkholes
---------
- /locatebiome minecraft:plains
- /tp @s <x> 100 <z>
- Fly surface and look for vertical shafts 20–50 blocks wide.

Blue Holes
----------
- /locatebiome minecraft:ocean
- /tp @s <x> 60 <z>
- Dive down Y=-20..-40.
- Look for near-perfect vertical shafts.

Tips
----
- /gamerule doDaylightCycle false
- /effect give @s minecraft:night_vision 9999 1 true
- /tp ~ ~100 ~ to scan from height
